# Todo Test 

Учебное приложение для управления задачами на Flutter.
Создано с целью изучения фреймворка Flutter, работы с состоянием и локальным хранилищем данных.

## Функционал 

### Основные возможности
* **Добавление задач:** Можно указать название и дату выполнения (через календарь).
* **Список задач:** Отображение списка с чекбоксами.
* **Сортировка:** Выполненные задачи автоматически перемещаются в конец списка, активные — сортируются по дате.
* **Редактирование:** Возможность изменить текст и дату существующей задачи.
* **Управление задачами:** Добавление названий, выбор даты выполнения через календарь.
* **Система приоритетов:** Задачи делятся на уровни (Low, Medium, High). Чекбоксы и элементы интерфейса динамически окрашиваются в зависимости от важности задачи.


### Управление данными, архитектура
* **Архив (Archive):**
    * Просмотр удаленных задач.
    * **Восстановление:** Вернуть задачу обратно в активный список.
    * **Полное удаление:** Стереть задачу навсегда.
* **Сохранение данных:** Все задачи (включая содержимое корзины) сохраняются локально (`через локальную БД ObjectBox`) и доступны после перезапуска приложения.
* **Множественное выделение (Multi-selection):** Режим выбора нескольких задач долгим нажатием для массового архивирования или удаления.
* **Контекстный AppBar:** Динамическая смена шапки приложения при переходе в режим выделения.
* **Интерактивные жесты:** Свайпы (Slidable) для быстрого редактирования, изменения даты, архивации или безвозвратного удаления.
* **Навигация:** Боковое меню (Drawer) для быстрого переключения между главным экраном и архивом.
* **State Management:** Логика вынесена из виджетов в отдельный сервис через ChangeNotifier и notifyListeners.
